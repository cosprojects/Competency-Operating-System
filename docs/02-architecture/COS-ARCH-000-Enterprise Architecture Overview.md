# Enterprise Architecture Goal (EN)

Enterprise Architecture provides a structured description of the platform from business concepts down to technological implementation.

Each architectural layer answers one specific architectural question.

Together these layers form a coherent architectural model.

---

# Цель Enterprise Architecture (RU)

Enterprise Architecture предоставляет последовательное описание платформы — от бизнес-концепций до технологической реализации.

Каждый архитектурный уровень отвечает на один конкретный архитектурный вопрос.

Все уровни совместно образуют единую архитектурную модель.
